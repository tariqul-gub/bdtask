STEP-BY-STEP SETUP GUIDE
-------------------------

1. INSTALL DEPENDENCIES
   Run these commands in the project root (c:\Exam\accounting):
   
   composer install
   npm install

2. CONFIGURE ENVIRONMENT
   The .env file should already exist. Verify these settings:
   
   DB_CONNECTION=mysql
   DB_HOST=127.0.0.1
   DB_PORT=3306
   DB_DATABASE=accounting
   DB_USERNAME=root
   DB_PASSWORD=

3. CREATE DATABASE
   Create a MySQL database named 'accounting':
   
   mysql -u root -p
   CREATE DATABASE accounting;
   EXIT;

4. RUN MIGRATIONS
   This will create all 7 tables:
   
   php artisan migrate

5. (OPTIONAL) SEED SAMPLE DATA
   To populate with sample branches, groups, and accounts:
   
   php artisan db:seed --class=AccountingSeeder

6. COMPILE ASSETS
   Build the frontend assets:
   
   npm run build
   
   OR for development with hot reload:
   
   npm run dev

7. START THE SERVER
   
   php artisan serve
   
   The application will be available at: http://localhost:8000

8. REGISTER/LOGIN
   - Visit http://localhost:8000
   - Click "Register" to create an account
   - Login with your credentials

9. START USING THE SYSTEM
   - Create branches
   - Create account groups
   - Create accounts
   - Create journal entries
   - View reports

================================================================================
  QUICK COMMANDS REFERENCE
================================================================================

# Clear all caches
php artisan cache:clear
php artisan config:clear
php artisan view:clear

# Run recurring entries command
php artisan accounting:run-recurring-entries

# Fresh migration (WARNING: Deletes all data)
php artisan migrate:fresh

# Fresh migration with seeding
php artisan migrate:fresh --seed --seeder=AccountingSeeder

# Run tests (if you create them)
php artisan test

================================================================================
  TROUBLESHOOTING
================================================================================

Problem: "Access denied for user" database error
Solution: Check DB_USERNAME and DB_PASSWORD in .env file

Problem: "Base table or view not found"
Solution: Run: php artisan migrate

Problem: Assets not loading
Solution: Run: npm run build

Problem: "Class not found" errors
Solution: Run: composer dump-autoload

Problem: Permission errors on Windows
Solution: Run terminal as Administrator
